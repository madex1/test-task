UMI.CMS позволяет строить структуры с любым уровнем вложенности. В самом верху показан домен, которым управляет система, под ним показано дерево разделов и подразделов сайта (домена).

Полная документация

Навигация

Это дерево также служит в качестве удобного и быстрого навигационного средства. Вы можете раскрыть или скрыть раздел при помощи пиктограмм и .

Раскрывая разделы, вы можете перейти к определенной новости, топику форума и даже товару в магазине.

Действия над объектами

При выделении объекта или строки появляется набор пиктограмм, отвечающих за возможные действия над этим объектом. (показать)

Просмотреть страницу на сайте.

Создать копию. При нажатии будет создана реальная копия страницы, редактирование которой никак не отразится на исходном оригинале.

Создать виртуальную копию. При нажатии на эту кнопку будет создана виртуальная копия объекта (или группы объектов). Она появится в самом низу того уровня дерева, где находится оригинал, к наименованию объекта добавится надпись "(виртуальная копия)". Виртуальная копия - это "ярлык" объекта (страницы). Т.е. если вы меняете информацию в одной из виртуальных копий объекта, она поменяется во всех. Это очень удобно в случаях, когда, например, необходимо поместить одну новость сразу в несколько лент новостей. Тогда при обновлении новости, она будет меняться сразу во всех лентах.

Страница видима посетителям сайта. Нажатие на иконку скроет страницу с сайта. (Она не будет удалена).

Страница скрыта. Нажмите, чтобы страница снова появилась на сайте.

Также при нажатии правой кнопкой мыши на объекте появится контекстное меню со списком возможных действий. При использовании браузера Opera для получения контекстного меню можно нажимать левую кнопку с зажатой клавишей Alt.

Групповые операции над объектами

Существует возможность совершать групповые операции над несколькими объектами системы. Для этого выделите несколько объектов (левой или правой кнопкой мыши — для отдельных объектов, или кнопкой мыши с зажатой клавишей Shift — для выделения группы объектов на одном уровне), после чего с помощью набора пиктограмм совершите необходимое действие.

Drag and Drop

Перемещать страницы и разделы можно мышью:

Наведите курсор на страницу, которую хотите переместить, и, удерживая левую кнопку мыши, перетащите страницу в нужный раздел. Если отпустить кнопку над названием другой страницы, перемещаемая страница станет дочерней.

Поиск по структуре

Над каждым деревом домена находится поле для поиска объектов.

Поиск осуществляется по вхождению строки. То есть, например, запрос "стран" покажет страницы со словами "страница", "иностранный", "страны". В простом режиме поиск ведется по всем полям объектов. В расширенном режиме можно выбрать поля, по которым будет осуществляться поиск и выбрать режим поиска: со всеми полями или хотя бы с одним из полей. Добавить поле можно, нажав на пиктограмму , удалить — с помощью пиктограммы .

Результаты поиска выводятся в виде дерева.
