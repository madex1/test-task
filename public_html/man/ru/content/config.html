Эта вкладка используется для управления шаблонами страниц.

Полная документация

Добавление шаблонов

Для добавления нового шаблона страницы нужно загрузить файл шаблона на сервер:

- в папку /xsltTpls/ (в случае xsl-шаблонов), или /tpls/content/ (в случае tpl-шаблонов).

или

- в директорию ~/templates/{имя_шаблона}/ (в случае использования нового способа хранения и управления шаблонами, который появился в системе с версии 2.8.5).

Затем необходимо зарегистрировать шаблон в системе: в этой форме нужно прописать название шаблона и имя загруженного файла и нажать «Сохранить».

Назначение шаблонов

При нажатии на пиктограмму «Изменить» вы попадете на страницу редактирования свойств шаблона.

Правила выбора шаблонов

При создании любых страниц выбор шаблона осуществляется по следующим правилам:

  • Если страница создается подразделом другой страницы, и у этого раздела еще нет ни одной дочерней страницы, — то в качестве шаблона по умолчанию выбирается шаблон этого раздела.
  • Если создается страница подразделом другой страницы, и у этого раздела есть одна или больше дочерних страниц, — то в качестве шаблона по умолчанию выбирается тот шаблон, который используется в большинстве подстраниц этого раздела.

Основной шаблон используется для системных страниц, таких как страница успешной отправки из формы обратной связи или страница результатов поиска.
