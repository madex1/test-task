Секция SEO настройки доменов предназначена для хранения и изменения настроек SEO для каждого домена системы.

Полная документация

В поле Префикс для TITLE можно задать префикс для заголовков окна. В поле TITLE (по умолчанию) задается заголовков окна для страниц, у которых не заполнено поле "title".

Вы можете вставлять значение поля H1 не только в конце, но в любом необходимом месте строки. Для этого следует использовать "макрос" %title_string%. Поместите его туда, где необходимо вставить значение поля H1 текущей страницы. например, конструкция "UMI.CMS :: %title_string% :: Современный интернет-магазин" на конкретной странице даст Title страницы "UMI.CMS :: DVD-диски :: Современный интернет-магазин".

В полях Keywords (по умолчанию) и Description (по умолчанию) можно указать мета-теги, которые будут подставлены по умолчанию, если соответствующие поля не заполнены у какой-либо страницы или публикации.

При включении настройки Регистрозависимый URL можно настроить действие при открытии url с отличным регистром букв.

При включении настройки Обрабатывать повторяющиеся слеши (/) в URL можно настроить действие при открытии url с лишними слешами.

При включении настройки Добавлять идентификатор страницы к alt-name дублирующих страниц к alt-name дублирующей страницы добавляется ее идентификатор.
