В модуле отображаются все слайдеры, доступные для публикации на сайте, и слайды, входящие в их состав.

Полная документация

В этом списке можно посмотреть все необходимые поля каждого слайда. Для добавления поля нужно кликнуть на пиктограмму со знаком плюс в правом углу. Для добавления нового слайдера наверху списка есть кнопка «Добавить слайдер».

Для того, чтобы добавить слайд, необходимо выбрать слайдер, в который необходимо добавить слайд. После этого вверху списка появится кнопка "Добавить слайд". Так же на этой странице доступно перетаскивание слайдов в слайдере, что позволяет изменить порядок их отображения.
