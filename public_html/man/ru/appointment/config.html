На вкладке настроек Настройки записи можно управлять настройками почтовых уведомлений

Полная документация

На данной странице можно включить/отключить любое из трех автоматических почтовых уведомлений модуля:

  • Уведомление администратора о новой записи;
  • Уведомление пользователя о сделанной заявке;
  • Уведомление пользователя об изменении заявки;

Для каждого типа уведомлений можно изменить шаблон заголовка и содержимого отправляемыx писем. В шаблонах доступны макросы, то есть метки, куда будут подставляться динамические данные (введенные пользователем поля). Для вставки макроса в шаблон нужно нажать на него в списке справа от редактируемого поля.
