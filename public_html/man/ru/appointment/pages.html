На вкладке Страницы можно увидеть список страниц, на которых размещен виджет онлайн-записи.

Полная документация

В этом списке можно посмотреть все необходимые поля каждой страницы. Для добавления поля в таблицу нужно кликнуть на пиктограмму с плюсом в правом углу.

Список страниц поддерживает быстрое редактирование свойств страниц. Для редактирования наведите курсор на необходимое свойство и нажмите на пиктограмму с карандашом, внесите изменения и нажмите enter.

В списке также можно отфильтровать список страниц по значениям полей. Для этого воспользуйтесь полями для ввода, которые расположены под именем колонки таблицы.

На данной странице можно добавить страницу. Для этого нужно нажать на большую синюю кнопку "Добавить страницу с онлайн-записью", которая расположена над таблицей.

Форма добавления страниц аналогична форме редактирования страницы.

Для редактирования страницы в отдельном окне нужно нажать на строку со страницей и выбрать в панели инструментов второй пункт. Там же можно удалить или отключить страницу (вторая и четвертая иконки в панели инструментов).
