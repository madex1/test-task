На вкладке Услуги можно увидеть список добавленных групп и услуг.

Полная документация

В этом списке можно посмотреть все необходимые поля каждой услуги. Для добавления поля в таблицу нужно кликнуть на пиктограмму с плюсом в правом углу.

Список услуг поддерживает быстрое редактирование свойств групп и услуг. Для редактирования наведите курсор на необходимое свойство и нажмите на пиктограмму с карандашом, внесите изменения и нажмите enter.

В списке также можно отфильтровать список услуг по значениям полей. Для этого воспользуйтесь полями для ввода, которые расположены под именем колонки таблицы.

Услуги можно перемещать между группами услуг.
