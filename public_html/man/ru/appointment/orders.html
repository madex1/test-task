Модуль позволяет организовать запись клиентов на услуги.

На вкладке Заявки можно увидеть список заявок, которые оставили пользователи.

Полная документация

В этом списке можно посмотреть все необходимые поля каждой заявки. Для добавления поля в таблицу нужно кликнуть на пиктограмму с плюсом в правом углу. Список поддерживает быстрое редактирование свойств заявки. Для редактирования наведите курсор на необходимое свойство и нажмите на пиктограмму с карандашом, внесите изменения и нажмите enter. Отредактировать можно поля "Статус", "Дата" и "Время".

В списке также можно отфильтровать список заявок по значениям полей. Для этого воспользуйтесь полями для ввода, которые расположены под именем колонки таблицы.
