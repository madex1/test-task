Модуль Опросы предназначен для проведения на сайте опросов и голосований.

Полная документация

На этой странице можно видеть список созданных опросов, хранящихся в системе

Здесь можно добавить новый опрос или редактировать уже существующие. Чтобы создать новый опрос, нажмите на ссылку Добавить опрос над списком. Откроется окно редактирования нового опроса. Если в системе доступны разные типы опросов, можно заранее выбрать тип опроса из списка, появляющегося при наведении курсора на ссылку.

Нажатие на ссылку с названием опроса откроет страницу редактирования его свойств.

Существующие опросы можно блокировать (скрывать с сайта).

Также вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
