Название опроса будет показано в списке голосований и в пути к странице. Подробнее об остальных полях читайте в топике Редактирование мета-информации раздела Редактирование страниц.

Полная документация

Активность. Если отмечена эта галка, посетители сайта будут видеть опрос на сайте. По умолчанию опрос активен.

Поле Вопрос предназначено для того, чтобы подробнее описать посетителям сайта сам вопрос.

Голосование закрыто. Если отмечена эта галка, то опрос становится закрытым, то есть посетители не смогут голосовать, но по-прежнему смогут видеть результаты опроса.

После сохранения опроса нужно будет дать посетителям варианты ответов. Для этого появится специальная область в свойствах опроса.
