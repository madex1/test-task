Данный модуль предназначен для создания страниц со скачиваемыми файлами. Здесь вы можете создавать страницы описания файлов, а затем помещать туда файлы, которые посетители сайта смогут скачивать. Модуль также ведет подсчет количества загрузок файлов посетителями.

Полная документация

В данном списке показаны страницы с файлами, уже доступными для скачивания. Здесь вы можете отредактировать страницу описания, удалить, либо сделать неактивной — таким образом файл будет недоступен для скачивания.

Чтобы добавить новую страницу с файлом, доступным для скачивания, нажмите на ссылку «Добавить файл».

Также вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
