Этот модуль предназначен для создания на сайте первичной базы знаний, именуемой «часто задаваемые вопросы». Подобные разделы на сайте значительно снижают нагрузку на отдел технической поддержки, так как в них содержатся ответы на наиболее популярные вопросы, которые посетителей могут интересовать.

Полная документация

Каждая база знаний (FAQ) представлена в системе UMI.CMS в виде проекта. Создав такой «проект», вы можете наполнять его списком часто задаваемых вопросов по определенной теме, на каждый из которых приводить ответ.

Созданные проекты располагаются в таблице. Поля таблицы можно добавлять самостоятельно, — для этого нужно нажать на пиктограмму и выбрать нужное поле. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов. Каждый из созданых проектов можно редактировать, скрывать с сайта и удалять.

Чтобы создать новый проект, нажмите на ссылку «Добавить проект» над списком проектов. Если в системе доступны разные типы проектов, можно заранее выбрать тип из списка, появляющегося при наведении курсора на ссылку.

Также вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
