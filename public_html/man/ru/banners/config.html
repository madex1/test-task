Полная документация

Количество показов, за которое нужно уведомить администратора об окончании срока показа баннера — в этом поле можно указать за сколько показов до отключения баннера следует уведомить администратора сайта по почте. Актуально для тех баннеров, которым выставлен параметр "Максимальное количество показов".

Не учитывать слова из поисковых запросов, по которым пользователь пришел на сайт, при обработке тегов баннера — позволяет отключить функциональность показа баннеров в соответствии с поисковым запросом, по которому пользователь пришёл на сайт.

При включенной опции Использовать модуль "Шаблоны уведомлений" для отправки писем письма администратору и клиенту о получении вопроса и ответе на него будут отправляться по шаблонам из модуля "Шаблоны уведомлений".
