Группа пользователей по умолчанию — группа пользователей, в которую пользователь попадает после регистрации на сайте.

Полная документация

Пользователь-гость — здесь можно задать пользователя из списка, чьи права будут использоваться для незарегистрированных пользователей. Эта настройка используется, если по каким-то причинам был удален исходный пользователь-гость. В других случаях использовать не рекомендуется.

Регистрация пользователей без активации — при включенной опции, аккаунт будет активирован автоматически после регистрации пользователя.

Проверять CSRF-токен при изменении настроек пользователя - если опция включена, то в форме редактирования параметров пользователя в его личном кабинете потребуется передать с формой csrf токен.

При создании пользователя/группы нужно устанавливать права на страницы сайта — если опция включена, при создании пользователя с правами, например на редактирование новостей, у каждой новости будут установлены права на редактирование для этого пользователя.

При изменении прав доступа пользователя/группы нужно изменять права на страницы сайта — если опция включена, то при добавлении/удалении прав, например, на редактирование новостей пользователя, у каждой новости будут установлены/удалены права на редактирование для этого пользователя.

При изменении настроек пользователя нужно запрашивать его пароль - если опция включена, то в форме редактирования параметров пользователя в его личном кабинете потребуется передать с формой текущий пароль пользователя, если вы хотите отредактировать логин, пароль или почтовый ящик пользователя.
