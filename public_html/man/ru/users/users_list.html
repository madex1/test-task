Модуль Пользователи позволяет создавать группы пользователей, которые, в зависимости от делегированных им прав, могут управлять контентом сайта через систему UMI.CMS: добавлять и редактировать новости, создавать новые страницы, других пользователей и т.п.

Полная документация

В этой вкладке представлена таблица со списком пользователей. В ней можно увидеть в какие группы входит тот или иной пользователь. Таблицу можно отсортировать и отфильтровать по любому из столбцов. Для добавления поля нужно кликнуть на пиктограмму в правом углу. Набор добавляемых полей задается базовыми типами модуля.

Поиск

Если список большой, найти определенного пользователя можно воспользовавшись строкой поиска, расположенной над списком.

Доступные действия

По аналогии с другими объектами системы UMI.CMS пользователей можно заблокировать, изменить данные (редактировать) или удалить: либо при помощи всплывающих пиктограмм, либо при помощи контекстного меню (правая кнопка мыши, или правая кнопка мыши + Alt для Opera).

Существует возможность совершать групповые операции над несколькими объектами системы. Для этого выделите несколько объектов (левой кнопкой с зажатой клавишей Ctrl), после чего с помощью контекстного меню совершите необходимое действие.

Также вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
