Идентификационные данные

Здесь вы можете задать Логин, Пароль и E-mail пользователя, а также внести его в какую-либо группу. Поля Логин, Пароль и E-mail являются обязательными для заполнения.

Вы можете внести пользователя в одну или несколько Групп пользователей, если в системе присутствуют таковые, либо, если вы уже добавляли группы Пользователей ранее. Для этого поставьте галочку напротив названия подходящей группы.

Информация в поле Провайдер регистрации появляется в случае если пользователь зарегистрировался на сайте при помощи интегрированного сервиса Loginza. При добавлении пользователя из админ-панели, данное поле заполнять не нужно.

Если вы не хотите чтобы данный пользователь мог пользоваться системой сразу после создания - снимите галочку Активирован. В этом случае учетная запись пользователя будет не активна. В дальнейшем вы можете активировать её в любой момент.

Дополнительная информация

Данный блок предназначен для применения некоторых настроек и добавления дополнительных данных о пользователе.

Ссылка "Оформить заказ от имени пользователя", механизм работы аналогичен одноименной ссылке при редактировании заказа.

Поля Зачисленные бонусы и Потраченные бонусы предназначены для организации системы бонусов в вашем интернет-магазине.

Поле Файловый менеджер позволяет указать, какой файловый менеджер будет доступен для пользователя. Поля Доступные директории (для файлового менеджера elFinder) и Дополнительные разрешенные типы файлов позволяют задать список доступных пользователю директорий и список типов файлов, разрешенных для загрузки пользователем, соответственно.

Поле Дата регистрации заполняется автоматически при регистрации пользователя через форму на сайте. При добавлении пользователя из админ-панели вы можете самостоятельно заполнить данное поле.

Персональная информация

При необходимости вы можете заполнить поля с личными данными пользователя: Фамилия и Отчество. Поле Имя является обязательным для заполнения.

Статистическая информация

Для удобства администраторов системы в данном блоке расположена информация об адресе, с которого пришел пользователь и адресе, на который пришел пользователь. Она заполняется автоматически при регистрации пользователя через форму на сайте.

Подробнее о настройке прав

Вы можете изменить набор прав, поставив дополнительные галочки. Следует иметь в виду, что права пользователя не могут быть меньше чем у пользователя "Гость", а также меньше чем права группы, в которую он входит.
