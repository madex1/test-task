Название сайта — используется в отчетах, при формировании RSS лент и др.

Полная документация

Доменный ключ — в этом поле отображается ваш текущий доменный ключ. Если вы используете версию UMI.CMS Start, то при переходе на платные лицензии, вам нужно будет удалить этот ключ. Появится сообщение о неверном лицензионном ключе с полем для ввода, в которое нужно будет ввести новый ключ, после чего иногда необходимо удалить файл /sys-temp/runtime-cache/registry на FTP и подтвердить изменение ключа.

Внимание: Ни при каких условиях, отличных от описанных выше, не удаляйте и не меняйте строку доменного ключа.

Разрешить браузерам кэшировать страницы — если эта опция включена, то браузер пользователя будет вначале проверять, изменилась ли страница, и если страница не менялась, то загрузит ее из локального кэша. При этом браузеру будет посылаться HTTP-заголовок Last-Modified, который содержит дату последнего изменения страницы. Эта опция поможет значительно снизить нагрузку на трафик и позволит быстрее открывать страницы.

Отключить автокоррекцию адресов — система анализирует набранный в адресной строке URL страницы сайта. Если в адресе присутствует ошибка, например, набран лишний или неверный символ, то автокоррекция откроет страницу, адрес которой наиболее похож на введенный. Поставьте галочку, если вы хотите отключить эту функцию.

Максимальный размер загружаемой фотографии — здесь вы можете указать максимальный размер загружаемой фотографии, задаваемый настройками PHP.

Максимальный размер загружаемого файла в PHP — выбирается системой, исходя из настроек PHP.

Разрешить совпадение адреса страницы с названием модуля — при включении этой функции становится возможным создание страницы с названием, совпадающим с названием модуля. К числу таких названий относятся news, catalog, faq, content и прочие. По умолчанию задание таких названий невозможно, и мы не рекомендуем включение данной функции.

Таймаут неактивности пользователя — позволяет указать максимальное время неактивности пользователя, после которого система попросит повторно ввести логин-пароль.

Занятое дисковое пространство — показывает дисковое пространство, занятое содержимым каталогов /files/ и /images/.

Ограничение дискового пространства — позволяет указать максимально возможный объем дискового пространства, которое выделяется для каталогов /files/ и /images/.

Отключить морфологический поиск — здесь вы можете отключить возможность поиска слова во всех его морфологических формах, а не только в заданном в строке поиска виде.

Отключить уведомление о большом количестве дочерних документов в структуре — позволяет отключить вывод предупреждения о большом количестве дочерних элементов при разворачивании структуры сайта.

Часовой пояс — предназначен для указания часового пояса в соответствии с требованиями сайта.

Модуль администратора по умолчанию — в выпадающем списке выберите модуль, в который пользователи будут попадать по умолчанию при переходе в режим администрирования. Если у какого-либо пользователя отсутствует доступ к выбранному модулю, то он будет перенаправлен на один из тех модулей, доступ к которым ему разрешен.
