Полная документация

Использовать настройки сайта — использовать ли для выбранного сайта указанные настройки.

E-mail отправителя — адрес для всей исходящей почты с вашего сайта. Это могут быть новостные рассылки, уведомления о поступившем товаре или снижении цены, подтверждение заказа и т.п.

Имя отправителя — получив письмо, адресат увидит эти данные в поле "От кого".

Имя отправителя — получив письмо, адресат увидит эти данные в поле "От кого".

Средство отправки писем — выбор средства отправки писем.

Отключить парсинг писем — отключает разбор тела письма. Отменяет:

  • Применение шаблона письма
  • Прикрепление внутренних изображений к письму
  • Замену относительных ссылок на абсолютные

Настройки SMTP:

Задержка — устанавливает задержку при отправке.

Хост — хост сервера.

Порт — порт сервера

Шифрование — выбор шифрования для отправки писем.

Включить авторизацию — включает/отключает авторизацию.

Логин — логин на сервере.

Пароль — пароль на сервере.

Включить режим отладки — включает/отключает режим отладки (используется для тестирования работы почты).

Использовать VERP — включает/отключает использование VERP.
