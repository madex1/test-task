На данной вкладке находятся настройки CAPTCHA. При включении этой функции во всех модулях, где используется защита от спам-ботов, будет показываться картинка с CAPTCHA.

Настройки делятся на два вида: общие для всех сайтов и специальные для каждого сайта (сайтом считается любая уникальная пара домен + языковая версия). Языковую версию можно поменять в правом верхнем углу административной панели.

Общие настройки действуют на всех сайтах, у которых не включена опция "Использовать настройки сайта). Если такая опция включена, то для данного сайта общие настройки игнорируются и используются только специальные настройки сайта.

Использовать настройки сайта — при включении этой настройки для сайта будут использоваться только его особые настройки (общие настройки будут игнорироваться).

CAPTCHA — Выбор механизма проверки CAPTCHA.

Доступные механизмы проверки:

  • Классическая — стандартная капча, на которой выводятся буквы и цифры.
  • Google reCAPTCHA — включает на сайте использование альтернативного механизма reCAPTCHA. Для использования этого механизма нужно сначала зарегистрировать сайт.

Класс отрисовки изображений CAPTCHA — с помощью этой настройки можно заменить стандартную отрисовку CAPTCHA. Синоним настройки "captcha.drawer" в файле config.ini. Если оставить поле пустым, будет использоваться класс по умолчанию "default".

Запоминать успешно пройденную пользователем CAPTCHA — при включении этой настройки пользователям не придется каждый раз вводить капчу в рамках одной сессии в браузере. Если пользователь один раз успешно ввел капчу, в последующих запросах ее уже не нужно будет заполнять.

Параметры sitekey и secret для reCAPTCHA — эти параметры нужно получить после регистрации сайта на сервисе Google reCAPTCHA.
