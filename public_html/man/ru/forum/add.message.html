Здесь вы можете задать следующие поля для нового сообщения:

Название сообщения — выводится на сайте над текстом сообщения

Активный — если отметку снять, посетители сайта не увидят данное сообщение

Сообщение — собственно текст данного сообщения, который можно изменить.

Описание остальных полей можно почитать в разделе Редактирование страниц.
