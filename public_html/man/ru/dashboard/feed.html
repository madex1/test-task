Этот модуль предназначен для просмотра событий, которые произошли на сайте в отсутствии пользователя.

Полная документация

События располагаются в ленте. Чтобы отфильтровать события по временному промежутку - укажите в заголовке ленты начальную и конечную дату, после чего нажмите кнопку Фильтровать

Доступные для отображения типы событий представлены в виде флажков на панели «Настройки» - отметьте интересующие Вас типы событий и нажмите кнопку Сохранить.
