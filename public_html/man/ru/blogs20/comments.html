На этой вкладке можно видеть список всех комментариев во всех блогах на сайте.

Полная документация

Для добавления поля нужно кликнуть на пиктограмму в правом углу. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов.

Также здесь можно скрыть, удалить или отредактировать любой комментарий, воспользовавшись всплывающими пиктограммами справа.

Так же вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
