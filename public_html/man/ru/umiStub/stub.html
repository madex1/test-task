Модуль "Доступ к сайту" позволяет управлять видимостью сайта из административной панели.

  • Использовать настройки сайта: Использовать ли для выбранного сайта отдельные настройки доступа к сайту.
  • Включить страницу-заглушку: Включает страницу-заглушку.
  • Отключить индексацию сайта: Отключает индексацию всех страниц сайта у поисковых роботов.
  • Заглушка: В данное поле вы можете добавить внешний вид страницы заглушки
