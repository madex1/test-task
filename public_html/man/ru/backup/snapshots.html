Полная документация

Если резервное копирование включено, то при каждом сохранении любой страницы вашего сайта будет создаваться новая копия, с помощью которой можно восстановить её предыдущую версию. Для этого предназначена секция Последние изменения, на странице редактирования объекта (новость, блог, страница контента и т.п.). На данной странице содержится список из ста последних изменений на сайте.
