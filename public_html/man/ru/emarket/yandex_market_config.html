Для интеграции с сервисом Яндекс.Маркет необходимо, чтобы ваш сайт работал через https, а также чтобы на сервере была установлена версия PHP не ниже 5.3.

Полная документация

Для подключения вашего Интернет-магазина к сервису Яндекс.Маркет произведите следующие настройки:

Регистрация нового приложения для обмена данными с сервисами Яндекса.

Перейдите по адресу oauth.yandex.ru/client/new, авторизуйтесь и заполните необходимые поля формы регистрации приложения.

В секции Права отметьте право "API Яндекс.Маркета для партнеров" (Яндекс.Маркет).

В поле Callback URI укажите адрес {имя_вашего_домена}/admin/emarket/yandexMarketTokenCallback/.

После сохранения вы будете перенаправлены на страницу вашего нового приложения, содержащую параметры Id приложения и Пароль приложения. Скопируйте их значения и заполните одноименные поля блока "Настройки для домена".

Нажмите кнопку Получить токен. Вы будете перенаправлены на страницу получения токена. После подтверждения поле Токен заполнится автоматически.

Заполните поле Логин (ваш логин на Яндексе) и нажмите Сохранить.

Регистрация интернет-магазина на Яндекс.Маркете.

Перейдите по адресу partner.market.yandex.ru, авторизуйтесь и зарегистрируйте ваш Интернет-магазин, заполнив обязательные поля формы.

Настройте новый экспорт, выбрав формат "Каталог в формате Яндекс.Маркет (yml)".

Для вновь созданной схемы экспорта нажмите Подготовить к экспорту для корректного преобразования данных, а затем Выполнить экспорт, в качестве необходимого результата укажите Получить ссылку на результат экспорта.

Полученную ссылку введите в поле для проверки параметров прайс-листа на втором шаге регистрации интернет-магазина и нажмите Проверить файл.

После успешной проверки вы будете перенаправлены в свой магазин на Яндекс.Маркете.

Перейдите в настройках в Параметры размещения и подключитесь к программе "Покупка на Маркете".

Перейдите в "Настройки" -> "Настройки API заказа" и заполните поля, указав:

URL API - https://{имя_вашего_домена}/emarket/yandex_market/

SHA1 fingerprint - SHA1 fingerprint хэш вашего SSL-сертификата

Тип авторизации - HEADER

Формат данных - JSON

После указания корректных данных вы можете получить свой Авторизационный токен и сохранить его в поле Токен магазина.

Заполните поле Номер кампании - восьмизначный код на главной странице вашего магазина в Яндекс.Маркете.

Также необходимо в файле config.ini (секция [seo]) задать значение параметра url-suffix.add равным "0".
