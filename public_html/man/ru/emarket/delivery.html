В этой вкладке перечислены способы доставки, доступные покупателям на сайте магазина.

Полная документация

Способы доставки можно отфильтровать по описанию и типу доставки.

Список также может быть экспортирован в формате CSV.

При добавлении нового способа доставки можно заранее выбрать тип из списка, появляющегося при наведении курсора на кнопку «Добавить».

Также вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
