На вкладке "Настройки параметров доставки" перечислены различные параметры, которые нужны для корректной работы доставки ApiShip.

Полная документация

  • Из каких полей товара брать значения для его характеристик - для формирования правильной информации о заказе в сервисе ApiShip нужно знать вес и размеры заказа. Вес заказа складывается из суммы всех весов составляющих его товаров. Для каждого из трех размеров заказа берется максимальный размер из товаров заказа.
  • Значения характеристик заказа по умолчанию - если у конкретного заказа не получится вычислить вес и размеры, система возьмет эти значения по умолчанию.
  • Значения характеристик заказа по умолчанию - если у конкретного заказа не получится вычислить вес и размеры, система возьмет эти значения по умолчанию.
