На этой странице можно управлять редиректами (перенаправлениями страниц).

Список редиректов может быть отсортирован и отфильтрован по любому из столбцов. Для сортировки нажмите на название столбца.

Быстрое редактирование редиректа:

  • нажмите на значок "карандаш" в поле, которое требуется отредактировать;
  • введите новое значение поля;
  • нажмите кнопку "enter" на клавиатуре.

Для добавления редиректа нажмите на большую синию кнопку "Добавить редирект".

Для редактирования редиректа выделите его (клик на название) и нажмите на первую иконку в панели инструментов модуля.

Для удаления редиректов выделите удаляемые редиректы (клик на название) и нажмите на вторую иконку в панели инструментов модуля.
