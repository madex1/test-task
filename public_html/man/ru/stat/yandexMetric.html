Если ранее вы уже пользовались сервисом Яндекс.Метрика и на подключаемом аккаунте у вас есть рабочие счетчики, то они будут отображены в списке счетчиков.

Полная документация

Для создания нового счётчика нажмите Добавить счетчик. На странице добавления необходимо задать название для счётчика и адрес сайта, на котором он будет установлен. После заполнения нажмите "Добавить счётчик". Счётчик добавится, и вы увидите новое поле - Код для вставки, содержащее код счётчика, который необходимо вставить в шаблон для всех страниц вашего сайта.

Cозданный счётчик появится в основном списке вкладки "Яндекс.Метрика". Нажмите на название счетчика, чтобы перейти к просмотру статистики, которую он собрал. Нажмите на статус счетчика, чтобы обновить статус.
