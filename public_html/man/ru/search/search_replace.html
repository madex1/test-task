На вкладке "Поиск и Замена" можно произвести поиск фразы и последующую ее замену во всех текстовых полях страниц системы.

Полная документация

Описание полей:

  • Найти — строка поиска в полях страниц.
  • Заменить на — строка замены найденных вхождений.
  • Где искать — типы страниц, по которым будет осуществляться поиск.

Если вхождения поисковой строки были найдены, то будет отображена таблица с результатами.

Описание полей таблицы результатов:

  • ID — внутренний идентификатор страницы.
  • Домен — домен, которому принадлежит страница.
  • Поле — название поля и его идентификатор, в котором были найдены вхождения поисковой фразы.
  • Имя страницы — название страницы, в полях которой были найдены вхождения.
  • Контент — часть содержимого поля страницы, в котором содержится вхождение поисковой фразы.
  • Редактировать — cсылка на редактирование страницы в административной части.
  • Заменить в тексте — можно отметить в каких полях страниц будет произведена непосредственная замена в тексте.
  • Заменить в ссылках — можно отметить в каких полях страниц будет произведена непосредственная замена в ссылках.
