Здесь вы можете видеть таблицу со списком разосланных сообщений. Для добавления поля нужно кликнуть на пиктограмму в правом углу. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов.

Полная документация

Нажатие на название сообщения в списке или на всплывающую пиктограмму напротив откроет страницу редактирования свойств сообщения.

Также вы можете редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
